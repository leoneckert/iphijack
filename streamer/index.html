ipiggyback

info

yes, piggybacking information onto the pixel values of an ip camera in real time.


Quick instructions

1. Click to inspect pixel values (why not start with xy: 18 5 ?).
2. Wait for messages to be decoded from the rgb values of a selected pixel.
3. Submit own message to be encoded and broadcasted *forever*.
4. Tell your friends which pixel to find your message on.
5. Find the ip camera, dance in front of it and manually encode a message.

project by Leon Eckert
